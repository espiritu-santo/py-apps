Please follow the instructions below to run the data explorer

1.Click Run! once without entering the X,Y coordinate pairs and a plot of the xyz data will be created.

2.The coords.txt file will be created in your Downloads folder. Copy this file and paste it into the static folder in the flaskapp directory.

3.From the figure use the mouse to hover over the starting position from which you would like to draw a cross-section.

4.With the mouse over the X,Y position take a screen shot (print screen) of the X,Y coordinates from the figure.

5.Do the same for the desired ending X,Y position.

6.Open the screen shots and this time **enter the X,Y coordinate pairs into the textbox.

7.Enter the starting X,Y coordinates and Press ENTER, followed by the ending X,Y coordinates.

Example:

#X1, Y1 = 7273961, 367077

#X2, Y2 = 7274257, 366641

8.Click Run!



**The starting X coordinate value must be less than the ending X coordinate value. The cross-section will be created from left-to-right.

The program was tested successfully using NAD 83 SPC X,Y coordinates in ift and ft. An example dataset has been provided with this app to get you started.
